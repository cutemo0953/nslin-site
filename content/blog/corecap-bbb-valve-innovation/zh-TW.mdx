## 問題：補胎液堵塞、打氣慢、還要多帶一個工具

### 補胎液逐漸堵塞氣嘴閥

使用無內胎輪胎的自行車騎士都面對過同一組問題，而這些問題全都指向同一個零件：valve core（氣嘴閥芯）。無內胎系統依賴液態補胎液來自動修補微小穿孔，但補胎液含有乳膠微粒，會在氣嘴閥內部逐漸堆積，讓每次打氣變得越來越費力。當空氣通道被部分堵塞，打氣需要更多次數的泵壓，充氣時間明顯拉長。

### 添加補胎液還要多帶一個工具

每次添加補胎液又需要拆卸 valve core，這意味著必須隨身攜帶一個獨立的氣嘴芯拆卸工具 — 一個很小、容易遺失、在座墊包底部很難找到的零件。這些問題看似瑣碎，但對自己維護輪胎的騎士來說，它們是每次補胎液添加和每次充氣都會遇到的反覆摩擦。

## 洞察：為什麼現有方案沒有真正解決問題

### 其他氣嘴帽只是頂開 valve core

市面上已經有「免工具」設計的氣嘴帽 — 轉開帽子就能打氣，不需要額外打開一個蓋子。但這些設計的打氣原理是透過氣嘴帽去**頂開**氣嘴桿內部的 valve core，讓空氣通過。問題在於 valve core 仍然留在氣嘴桿裡面。

### Valve core 還是擋在裡面

這顆 valve core 包含彈簧和柱塞機構，佔據了氣嘴桿內相當大的截面積。無論帽子的設計多巧妙，只要 core 還在裡面，空氣通道就被壓縮，流量就有上限，補胎液殘留物也更容易在 core 周圍的窄縫堆積，加劇堵塞。換句話說，現有的「免工具」設計改善了操作便利性，但完全沒有解決流量瓶頸的根本原因。

## 解法：把閥門做進帽裡，讓氣嘴桿內部完全淨空

### 全新機構，從零設計

BBB CoreCap 由位於台南的奕道實業（N.S.-LIN）設計製造，由荷蘭自行車零件品牌 <a href="https://bbbcycling.com/en_en/corecap" target="_blank" rel="noopener noreferrer">BBB Cycling</a> 獨家銷售。它用一個完全不同的方式解決了上述問題。**CoreCap 將閥門機構（單向閥功能）直接整合進氣嘴帽中。** 這個全新的機構設計（已申請專利）意味著氣嘴桿內部不再需要傳統的 valve core — 可以直接拔除。

### 氣嘴桿內部完全淨空

拔除之後，氣嘴桿內部完全淨空：沒有彈簧、沒有柱塞、沒有任何阻礙氣流的零件。空氣通道的截面積最大化，打氣速度大幅提升，即使補胎液有殘留，開放的內部空間也遠比有 core 的窄通道更難被堵塞。

### 帽就是工具

氣嘴帽本身就是 valve core 的拆卸工具 — 需要添加補胎液時，轉下帽子、用帽子拔出 core，完成，不需要額外工具。一個零件同時解決了兩個問題：提升流量（移除 core）和消除額外工具（帽即工具），用更少的零件而不是更多。這不是對現有設計的漸進式改良，而是一個全新的產品架構，從零開始設計。

## 工程驗證：為什麼這比聽起來難得多

把閥門做進帽裡的概念聽起來直觀，但要在量產品質下實現，涉及多重互相矛盾的工程約束。

### 流量、強度與氣密的三角衝突

移除 valve core 釋放了內部空間，但要進一步提升流量，仍然需要盡可能減薄氣嘴桿的壁厚（外徑受限於輪圈氣嘴孔標準）。壁厚減少直接衝擊結構強度和氣密性兩個指標。桿身必須承受打氣泵施壓、輪圈鎖緊螺母夾持力、騎乘震動，任何微小變形都可能導致螺紋失效或密封面偏移。同時，無內胎系統最怕慢漏氣，壁越薄，每一個密封面的加工精度要求越高 — 底座接合處、帽內的閥門機構、鎖緊螺母壓合面，每個接觸面都是潛在的洩漏路徑。**高流量、結構強度、長期氣密構成工程三角衝突。** CoreCap 必須在這三者之間找到最佳平衡點，而且是在一個沒有產業先例可參考的全新機構上。

### 氣嘴帽的多重功能面

帽本身的精度要求也極高。同一顆帽必須同時容納與氣嘴桿配合的螺紋面（騎乘中不鬆脫）、夾持 valve core 進行拆卸的內部輪廓，以及新的閥門機構（單向閥功能）。這些功能面在極小的空間內緊鄰排列，它們之間的相對位置和尺寸精度直接決定產品成敗，而且必須在數千次反覆使用後仍然可靠。

### 為什麼 CNC 是唯一選擇

CoreCap 是全新機構，所有 CNC 程式和治具都從零開發 — 這本身就排除了大多數量產工藝。塑膠射出成型的模具收縮率變異會讓 valve core 夾持功能失效；鋅合金壓鑄在反覆受力後容易疲勞裂紋；沖壓無法實現帽內部的複雜幾何。CoreCap 全系列採用 **6061-T6 鋁合金 CNC 車削加工**，實現壁厚精密控制（在最大化流量的同時維持結構強度）、單次裝夾完成內外螺紋的同軸度（避免二次裝夾偏差），以及遠優於鑄造或射出件的密封面粗糙度。材料特性也不可替代：6061-T6 抗腐蝕（補胎液含氨和乳膠）、耐疲勞（T6 熱處理），且不會增加車輪的感知重量。

### 168+ 小時氣密驗證

全新機構沒有歷史數據可參考，氣密性必須靠實測建立信心。**每批出貨以抽樣方式進行超過一週（168 小時以上）的靜態保壓測試，要求零可量測壓力衰減。** 168 小時是最低門檻而非終點，實際測試時間往往更長。判定標準是絕對的：不是「洩漏率低於閾值」，而是在測試精度內完全不漏氣，而且驗證的是整支氣嘴閥的系統氣密性，不只是單一密封面。為什麼這麼嚴格？因為無內胎系統的失效模式和有內胎系統根本不同。一旦氣嘴閥微滲漏，補胎液沿洩漏路徑固化，在密封面形成不均勻殘留物，反而加速密封劣化。**「幾乎不漏」和「完全不漏」是兩個截然不同的品質等級** — 直接影響品牌的保固成本與終端使用者信賴度。

### 陽極處理

CoreCap 提供六種陽極處理顏色：黑、藍、古銅、金、紅、紫。陽極處理在鋁合金表面生成緻密的氧化鋁膜（Al₂O₃，10-25 μm），硬度（HV 300-400）遠高於基材（HV 95-110），提供耐磨和抗腐蝕保護。與烤漆不同，陽極膜是表面的化學轉化層，不會剝落。

## iF 設計獎

CoreCap 榮獲 **iF 設計獎**，這是國際最具聲望的工業設計競賽之一。評審肯定了產品在形式與功能上的整合 — 用更少的零件解決真實的使用者問題。

**最好的氣嘴閥創新，不是堆疊功能，而是消除不必要的零件。**

## OEM 合作：從概念到量產的完整能力

CoreCap 不是從其他製造商授權而來。從機構設計、專利申請、CNC 程式開發、治具設計，到氣密測試和量產出貨，全部由奕道在台南統籌完成，部分製程委託在地協力廠。這個端到端的能力對所有品牌合作夥伴開放。

### 客製化設計

客製化是奕道的核心強項。CoreCap 本身就是客製化開發的案例 — 全新專利機構，從概念到量產一手包辦。無論是客製桿長、特殊顏色、品牌雷射雕刻，或是全新功能整合設計，研發團隊都能從初始概念參與。
